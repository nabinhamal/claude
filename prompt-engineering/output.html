### Extracting Topics from a Scholarly Article

To extract topics from a passage of text, we can use Natural Language Processing (NLP) techniques such as keyword extraction or topic modeling. Here, we will use a simple keyword extraction approach to identify the main topics.

### Code

```python
import json
import re
from collections import Counter

def extract_topics(text):
    # Remove punctuation and convert to lowercase
    text = re.sub(r'[^\w\s]', '', text).lower()
    
    # Split text into words
    words = text.split()
    
    # Count word frequencies
    word_freq = Counter(words)
    
    # Identify common topics (.keywords)
    common_words = ['the', 'and', 'a', 'is', 'for', 'of', 'in', 'to']
    topics = [word for word, freq in word_freq.items() if word not in common_words and freq > 1]
    
    # Sort topics alphabetically
    topics.sort()
    
    return topics

text = """
Environmental sustainability is crucial for the future, with climate change being a significant concern. 
The shift towards renewable energy sources, such as solar and wind power, is vital for reducing greenhouse gas emissions and mitigating the impacts of climate change. 
Governments and corporations are investing heavily in renewable energy technologies to create a more sustainable future.
"""

topics = extract_topics(text)

# Convert topics to JSON array
json_topics = json.dumps(topics)

print(json_topics)
```

### Explanation

This code defines a function `extract_topics` that takes a passage of text as input, removes punctuation, converts to lowercase, splits the text into words, counts the word frequencies, and identifies the common topics (keywords) by filtering out common words (e.g., "the", "and") and selecting words with a frequency greater than 1. The resulting topics are sorted alphabetically and returned as a list. The list is then converted to a JSON array using the `json.dumps` function.
